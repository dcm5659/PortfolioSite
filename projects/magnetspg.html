Project Description

Poem Fridge is our take on a magnetic poetry app. It allows the user to create their own poetry using an interface similar to magnets on a fridge. There are a number of different word lists and fonts to choose from. The users can then screenshot and share their poems using the share button. They can also edit the background of the app using the load function.
