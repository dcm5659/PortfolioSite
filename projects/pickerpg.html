Project Description

PC Part Picker is my attempt to make PC building simpler for beginner PC builders. It utilizes the Best Buy and Google Maps API's to locate the closest Best Buy location to buy the part the user chooses. It searches using a simple drop down menu to make sure the user doesn't miss out on any key parts.
